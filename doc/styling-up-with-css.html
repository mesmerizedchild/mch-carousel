MCh Carousel: Styling up with CSS

Here you will find information on how to style up your MCh Carousel. If this page is your very first encounter with MCh Carousel, you might want to read the Quick Start first and take if from there [you will eventally land back onto this page...]

Please note that, in order to display this page and other documentation pages correctly, you will need an active Internet connection, as some resources are downloaded live from the net.

The DOM structure of MCh Carousel at run-time

MCh Carousel reorganises the DOM heavily, turning the simple div element with a handful of child img elements, into an articulated DOM tree [or a lush DOM forest, if you prefer...]

Understanding the DOM structure is essential to styling MCh Carousel, as it's not immediately obvious where each piece of information [that is, captions, buttons, hyperlinks, etc] ends up at run-time.

ID and class

If you specify an ID or a class for your images, these will be picked up by MCh Carousel. For example:

  <img src="images/image1.jpg"
       id="image-1" />
  <img src="images/image2.jpg"
       class="even-image" />
  <img src="images/image3.jpg"
       id="image-3" class="odd-image" />

Note that the ID and/or class that you specify in the img tag will be "moved" to another element at run-time, but in your CSS you certainly may rely on these ID's and classes to change the appearance of one single image, or a group of images. More information below.
