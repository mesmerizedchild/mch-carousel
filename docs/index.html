MCh Carousel Quick Start

Required libraries

Before you can use MCh Carousel, you need to add the following to your HTML:

  <script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/2.1.4/jquery.min.js"></script>
  <script type="text/javascript" src="https://cdnjs.cloudflare.com/ajax/libs/jquery-mousewheel/3.1.13/jquery.mousewheel.min.js"></script>
  <script type="text/javascript" src="mch-carousel.min.js"></script>
  <link rel="stylesheet" href="mch-carousel.css">

The first line includes jQuery, which is an absolute requirement; without it, MCh Carousel just won't work.

The second line includes jQuery Mouse Wheel Plugin, which is not strictly necessarly, but will guarantee cross-browser support for mouse wheels, trackpads and Apple's Magic Mouse.

The third line loads MCh Carousel, and finally the fourth line loads the core MCh Carousel styling sheet [i.e. the very basic CSS without which the carousel would display a mess...]

Styling MCh Carousel up

CSS styling is at the core of what MCh Carousel looks like.

You may start off by linking one of the themes that come along with the code, in the themes directory, for example:

  <link rel="stylesheet" href="themes/mch-carousel-basic-theme.css">

You will also have to specify the height of the carousel. For example you could embed the following in your HTML to make the carousel 393 pixels tall:

  <style type="text/css">
    #mch-viewport {
      height: 393px;
    }
    .mch-image-container {
      height: 393px;
    }
  </style>

Note the double declaration, where the heights must match. If you will be using LESS for your development, a mix-in is available that does the double declaration for you [and works with media queries too!] See Using LESS for further details.

Listing the images for MCh Carousel

You now need to list the images of the carousel by using the <img> tag, wrapped in two special <div>'s, as in the following example:

  <div id="my-carousel"><!-- this div may have whatever id and/or class you want -->
    <div id="mch-image-list"><!-- on the other hand, this div must have its id set to "mch-image-list", and no class, nor styling -->
      <img src="images/image1.jpg" />
      <img src="images/image2.jpg" />
      <img src="images/image3.jpg" />
      <img src="images/image4.jpg" />
      <img src="images/image5.jpg" />
      <img src="images/image6.jpg" />
    </div>
  </div>

Ensure that the sum of the widths of your photos exceeds the width allotted to the carousel on your page, otherwise they will just show side by side, no sliding. And, for that, you need not complicate your life with a carousel :) ...

Starting MCh Carousel up

For the carousel to appear [and begin to automatically slide images for you], you need to add the following:

  <script type="text/javascript">
    $(window).load(function() {
      $('#my-carousel').mchCarousel(); /* This one line starts the carousel up */
    });
  </script>

That's it! Done!

At the moment, though, the carousel is still missing some features: no captions, no hyperlinks, no on-mouse-over images. Read on to TODO in order to learn how to add them.
